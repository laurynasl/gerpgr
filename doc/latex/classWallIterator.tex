\subsection{Detailed Description}
Let x and y be coordinates so that 0$<$=x$<$width, 0$<$=y$<$height. Then if x==0 or x==(width-1) or y==0 or y==(height-1), it is a wall. Wall\-Iterator iterates these walls.
